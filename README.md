# mexicombat
Fighter game in Haxepunk for the upcoming EP release for http://cheesesofmexico.com

# Requirements
Openfl


# Install
Clone project and make a project.xml. 

  git clone https://github.com/fritzvd/mexicombat/
  cd mexicombat
  cp project.xml.example project.xml
  lime test neko
